# Маршруты приложения Cloud.co

## Основные страницы

### Главная страница
- **Маршрут**: `/`
- **Компонент**: `HomePage`
- **Описание**: Главная страница с поиском и карточками услуг

### Поиск
- **Маршрут**: `/search`
- **Компонент**: `SearchPage`
- **Описание**: Поиск по адресу, специалистам и услугам

### Пустой поиск
- **Маршрут**: `/empty-search`
- **Компонент**: `EmptySearch`
- **Описание**: Экран для пустых результатов поиска

## Оборудование

### Каталог оборудования
- **Маршрут**: `/equipment`
- **Компонент**: `EquipmentCatalog`
- **Описание**: Список всего доступного оборудования

### Карточка оборудования
- **Маршрут**: `/equipment/[id]`
- **Компонент**: `EquipmentDetail`
- **Описание**: Детальная информация об оборудовании с 3D поддержкой

### Бронирование оборудования
- **Маршрут**: `/booking/[equipmentId]`
- **Компонент**: `BookingPage`
- **Описание**: Пошаговый процесс бронирования:
  1. Выбор файлов для загрузки
  2. Выбор типа аренды (дистанционное/личное)
  3. Выбор даты и времени
  4. Подтверждение

### После аренды
- **Маршрут**: `/post-rental`
- **Компонент**: `PostRentalPage`
- **Описание**: Управление файлами после завершения аренды

## Учреждения и курсы

### Карточка учреждения
- **Маршрут**: `/institution/[id]`
- **Компонент**: `InstitutionDetail`
- **Описание**: Информация об учреждении, контакты, направления обучения

### Детальная страница курса
- **Маршрут**: `/course/[id]`
- **Компонент**: `CourseDetail`
- **Описание**: Информация о курсе, календарь для записи на консультацию

## Файловая система

### Файловая система
- **Маршрут**: `/filesystem`
- **Компонент**: `FileSystemPage`
- **Описание**: Управление файлами проекта, загрузка/скачивание

## Возможности платформы

### Страница возможностей
- **Маршрут**: `/features`
- **Компонент**: `FeaturesPage`
- **Описание**: Описание возможностей платформы, преимуществ удаленного управления и файловой системы

## Служебные страницы

### Экран загрузки
- **Маршрут**: Автоматически (через Suspense)
- **Компонент**: `LoadingScreen`
- **Описание**: Экран загрузки с логотипом

## Навигация

Все страницы доступны через:
- Прямые ссылки
- Кнопки навигации в компонентах
- Header с кнопкой "Назад"
- Кнопка "Файлы" в Header (на главной странице)

## Интеграция

Все маршруты используют Next.js App Router и поддерживают:
- Динамические параметры (`[id]`, `[equipmentId]`)
- Query параметры для поиска
- Навигацию через `useRouter` из `next/navigation`
