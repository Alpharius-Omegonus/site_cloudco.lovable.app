# Changelog

## [0.1.0] - 2025-01-XX

### Добавлено

#### Основная структура
- ✅ Настроен Next.js 14 проект с TypeScript
- ✅ Создана структура папок (app router, components, screens, styles)
- ✅ Настроены конфигурационные файлы

#### Экраны
- ✅ **Экран загрузки** - с логотипом Cloud.co- и анимацией
- ✅ **Главная страница** - поиск и карточки услуг
- ✅ **Каталог оборудования** - список оборудования с фильтрацией
- ✅ **Карточка оборудования** - детальная информация с поддержкой 3D
- ✅ **Поиск** - по адресу, специалистам, услугам
- ✅ **Пустой поиск** - экран для пустых результатов
- ✅ **Карточка учреждения** - информация об учреждении
- ✅ **Детальная страница направления/курса** - с календарем и бронированием

#### Компоненты
- ✅ **Header** - навигация с логотипом, уведомлениями и профилем
- ✅ **SearchBar** - поисковая строка
- ✅ **ServiceCard** - карточки услуг
- ✅ **EquipmentCard** - карточки оборудования
- ✅ **SearchResultCard** - карточки результатов поиска
- ✅ **Model3D** - компонент для 3D моделей
- ✅ **Calendar** - переиспользуемый компонент календаря
- ✅ **LoadingScreen** - экран загрузки

#### Функциональность
- ✅ Навигация между всеми экранами
- ✅ Поиск с фильтрацией
- ✅ Бронирование консультаций с выбором даты и времени
- ✅ Поддержка 3D моделей (готово к интеграции Three.js)
- ✅ Темная тема
- ✅ Адаптивный дизайн

#### Стили
- ✅ Темная тема с градиентами
- ✅ Шрифты: Onest (Google Fonts) и CalvinoDisplayItalic (локальный)
- ✅ CSS модули для всех компонентов
- ✅ Адаптивный дизайн для мобильных устройств

### Технические детали

- Использован Next.js 14 App Router
- TypeScript для типобезопасности
- CSS Modules для стилизации
- Компонентный подход

### Известные ограничения

- Данные hardcoded (требуется интеграция с бэкендом)
- 3D модели требуют интеграции Three.js для полноценной работы
- Шрифт CalvinoDisplayItalic требует добавления файла
- Изображения требуют добавления в папку public/
